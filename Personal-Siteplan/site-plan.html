Images for the Home page

[Page 2]


Creator
On the second page I plan to have a character creator that takes in all the important info and then saves it as a json or another file type to be loaded later. That way the user can take all the info they learned on the first page and apply it
Notes
In the character creator there will also be notes at each section explaining what each stat could be used for so that when someone is picking certain things they can make a more informed decision
an example would be next to the Ability Scores:
T

he best way to think of it is to think in terms of a tomato
Strength is being able to crush a tomato.
Dexterity is being able to dodge a tomato.
Constitution is being able to eat a bad tomato.
Intelligence is knowing a tomato is a fruit.
Wisdom is knowing not to put a tomato in a fruit salad.
Charisma is being able to sell a tomato based fruit salad
